# AcadBot
Its an AI-powered academic advising chatbot using low-code tools integrated with conversational AI APIs. The Bot provides course recommendations, academic planning suggestions, and automated appointment scheduling.
Created and hosted in CodeSandbox 
direct URL :https://8vxczw-3000.csb.app/
Features of Chatbot : -  
  •	Chat Tab — Ask anything: “recommend AI courses”, “attendance policy”, “book appointment”
	•	Analytics Tab — See live sentiment and topic analytics update as you chat
	•	Appointments Tab — View and manage all booked advisor sessions
	•	Feedback Tab — Submit feedback with real-time sentiment detection
